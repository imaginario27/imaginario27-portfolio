## Gallery
- Justifing masonry means justifying the last row, not all. Row images should fill the entire width of the container, but the last row may be shorter if there are not enough images to fill it. This is the default behavior for masonry layouts, and it allows for a more natural and visually appealing arrangement of images. Justifying all rows in a masonry layout can lead to awkward spacing and uneven image sizes, which can detract from the overall aesthetic of the gallery.
- Rows in masonry should always share the same height, but the width of the images can vary. This creates a more cohesive and visually appealing layout, as it allows for a consistent flow of images while still accommodating different aspect ratios and sizes. Varying the height of rows in a masonry layout can lead to a disjointed and chaotic appearance, which can be distracting for viewers.
